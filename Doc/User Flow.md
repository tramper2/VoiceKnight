# User Flow: Voice Knight

1. **진입:** 사용자가 GitHub Pages URL 접속.
2. **권한 요청:** 'Game Start' 버튼 클릭 시 마이크 사용 권한 팝업 노출.
3. **튜토리얼:** "점프라고 말해보세요" 문구와 함께 음성 테스트 단계 진행.
4. **메인 게임 루프:**
    - 캐릭터 자동 달리기 시작.
    - 음성 입력("점프") -> 캐릭터 도약 -> 장애물 회피.
    - 음성 입력("공격") -> 무기 발사 -> 적 제거.
5. **게임 오버:** 장애물 충돌 시 최종 점수 표시 및 '다시 하기' 버튼 노출.
