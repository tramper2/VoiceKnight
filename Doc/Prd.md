# 프로젝트명: Voice Knight (가제) - 음성 제어 횡스크롤 액션 게임

## 1. 제품 개요
사용자의 음성을 실시간으로 인식하여 캐릭터를 조작하는 웹 기반 횡스크롤 액션 게임. 키보드 조작이 어려운 상황이나 새로운 게임 경험을 원하는 사용자에게 몰입감을 제공함.

## 2. 핵심 가치 (Core Value)
* **Hands-Free:** 오직 목소리만으로 점프하고 공격하는 재미.
* **Intuitive:** 직관적인 명령어("점프", "공격")로 즉각적인 피드백 제공.
* **Accessibility:** 웹 브라우저만 있으면 어디서든 실행 가능.

## 3. 타겟 사용자
* 새로운 조작 방식의 인디 게임을 즐기는 게이머.
* 웹 기술 기반의 음성 인식 인터페이스 구현에 관심이 있는 개발자.
